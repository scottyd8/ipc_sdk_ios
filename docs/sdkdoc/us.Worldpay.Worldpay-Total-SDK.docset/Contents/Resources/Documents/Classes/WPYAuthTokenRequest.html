Overview

This object is sent when requesting an authorization token from the server so that other APIs can be used. The SDK automatically stores the token in the device keychain, but returns a copy of the token just in case the developer needs it for some reason. Only one ID and Key/PIN should be set, based on the processing gateway the application is to use
