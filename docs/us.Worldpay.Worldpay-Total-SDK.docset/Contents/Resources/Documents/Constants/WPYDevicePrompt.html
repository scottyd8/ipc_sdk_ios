WPYDevicePrompt

When certain text needs to be displayed to the user, these prompts will be sent to the application developer to indicate what kind of text should be displayed. For terminals that canDisplayText, you can choose not to handle device prompt text if you’d like to use the default text. For terminals that cannot display text, all of these prompts must be handled and properly presented to the card holder
